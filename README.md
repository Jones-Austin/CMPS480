## Module 2B – Working Interface with Mock Server

### Features Implemented
- Product listing with images
- Search by product name
- Filter by category
- Sort by:
  - Price (Low → High)
  - Price (High → Low)
  - Rating (High → Low)
- Simulated backend interaction using hardcoded JSON data

### Data Source
The product dataset used in this module matches the mock backend API located in:
MockAmazon/server/index.js

This ensures consistency between frontend and backend modules.
