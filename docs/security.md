## How will we keep the system secure?

- We will validate user input on both the frontend and backend.
- Only authorized users will be able to access protected features.
- Sensitive information will not be exposed publicly.

---

## What data is sensitive?

- User passwords
- User email addresses
- Login/session information
- Order or cart data

---

## How will we protect sensitive data?

- Passwords will not be stored in plain text.
- Access to sensitive data will require authentication.
- We will avoid storing unnecessary personal information.

---

## What technologies will we use?

- Secure password storage 
- Backend validation
- Secure communication methods supported by our hosting platform
